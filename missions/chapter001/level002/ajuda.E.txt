\b;Objective

Move the robot to the target using a program created by you.

\t;Instructions

To create a program click in the plus button in the lower left corner of your screen;
To edit the program code, select it and click in the button with the keys {..};
To run the program, click the button with an "arrow".

The following functions will be used in this mission:
   \c;\l;move\u cbot\move;\n; - moves forward or backward;
   \c;\l;turn\u cbot\turn;\n; - rotates according to the given angle.
